## 📋 Overview 📖
**QuickStay** is a modern full-stack hotel booking platform that allows users to discover, book, and manage hotel rooms effortlessly.  

🏨 Hotel owners can:
- 📌 List properties  
- 📦 Manage bookings  
- 💰 Track revenue via dashboard  

---

## ✨ Features 🌟

### 👤 For Users 🧑‍💻
- 🔍 Search and filter rooms  
- 📅 Real-time availability  
- 💳 "Pay at Hotel" option  
- 📱 Mobile responsive design  
- 🖼️ Room image galleries  
- 📋 Booking management  

### 🏢 For Hotel Owners 🧑‍💼
- 📊 Analytics dashboard  
- 🏨 Manage listings  
- 🖼️ Upload images  
- ✅ Availability control  
- 📈 Booking history

### ⚛️ Frontend 💻
- ⚛️ React 19 (Vite)  
- 🎨 TailwindCSS  
- 🧭 React Router DOM v7  
- 🔔 React Hot Toast  
- 📦 Lucide React & React Icons  

### 🟢 Backend 🖥️
- 🟢 Node.js + Express  
- 🍃 MongoDB + Mongoose  
- 🔐 Clerk Authentication  
- 📧 Nodemailer

- 💳 Stripe (Test Mode)
